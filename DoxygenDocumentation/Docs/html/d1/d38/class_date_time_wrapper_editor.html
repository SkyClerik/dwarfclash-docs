Подробное описание

Пользовательский редактор для DateTimeWrapper и TimeSpanWrapper, предоставляющий удобный интерфейс для редактирования даты и времени в инспекторе Unity с использованием UI Toolkit.

Этот редактор заменяет стандартное отображение [Serializable] полей на более структурированный набор полей для года, месяца, дня, часа, минуты и секунды.
