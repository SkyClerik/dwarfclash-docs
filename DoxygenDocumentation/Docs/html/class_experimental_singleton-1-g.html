Detailed Description

Экспериментальная реализация паттерна 'Одиночка' (Singleton) для MonoBehaviour, которая автоматически создаёт и сохраняет единственный экземпляр объекта между сценами.

Template Parameters
TТип компонента, для которого реализуется 'Одиночка'. Должен быть наследником MonoBehaviour.

Для использования, унаследуйте ваш класс от ExperimentalSingleton<YourClassName>. Экземпляр будет доступен через статическое свойство YourClassName.Instance. Этот синглтон пытается найти существующий экземпляр в загруженных сценах перед созданием нового. При выходе из приложения устанавливается флаг, чтобы предотвратить создание 'призрачных' объектов.
