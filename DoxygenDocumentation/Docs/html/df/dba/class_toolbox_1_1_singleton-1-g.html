Потокобезопасная и надежная реализация паттерна 'Одиночка' (Singleton) для MonoBehaviour. Гарантирует наличие только одного экземпляра объекта и его сохранность между сценами. Подробнее...

Базовые классы:UnityEngine.MonoBehaviour.

Производные классы:BattleTimePage, BattlefieldManager, BuildingsPage, CustomRendererManager, DataHandler, Gameplay.UI.UIPage< T >, Gameplay.UI.UserInterfaceShare, GlobalBackgroundSound, InstancingManager, Player, SelfHudPage, UI.BattleTablePage, UI.GameHudPage, UI.HeroPage и UserInterfaceRaycaster.

Защищенные члены

 Обрабатывает логику 'Одиночки' при пробуждении объекта. Гарантирует, что существует только один экземпляр, и уничтожает дубликаты. Делает основной экземпляр "бессмертным" между сценами. Не забываем protected override void Awake() и вызов base.Awake();.

 Возвращает единственный потокобезопасный экземпляр 'Одиночки'. Если экземпляр не существует, он будет найден на сцене или создан новый.

Подробное описание

Потокобезопасная и надежная реализация паттерна 'Одиночка' (Singleton) для MonoBehaviour. Гарантирует наличие только одного экземпляра объекта и его сохранность между сценами.

Параметры шаблона
TТип компонента, для которого реализуется 'Одиночка'. Должен быть наследником MonoBehaviour.

Для использования, унаследуйте ваш класс от Singleton<YourClassName>. Экземпляр будет доступен через статическое свойство YourClassName.Instance.

Согласование типов
