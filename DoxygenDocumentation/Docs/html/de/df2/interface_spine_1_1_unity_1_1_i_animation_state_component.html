Интерфейс Spine.Unity.IAnimationStateComponent

A Spine-Unity Component that uses a Spine.AnimationState to animate its skeleton. Подробнее...

Базовые классы:Spine.Unity.ISpineComponent.

Производные классы:Spine.Unity.SkeletonAnimation и Spine.Unity.SkeletonGraphic.

Свойства

AnimationState AnimationState [get]
 Gets the Spine.AnimationState of the animated Spine Component. This is equivalent to SkeletonAnimation.state.
bool UnscaledTime [get, set]
 If enabled, AnimationState time is advanced by Unscaled Game Time (Time.unscaledDeltaTime instead of the default Game Time(Time.deltaTime). to animate independent of game Time.timeScale. Instance SkeletonGraphic.timeScale and SkeletonAnimation.timeScale will still be applied.
