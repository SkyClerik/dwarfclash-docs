◆ RemoveChildComponentsRecursive()

void Toolbox.TransformExt.RemoveChildComponentsRecursive ( this Transform obj,
params Type[] excludeTypes )
inlinestatic

Удаляет все компоненты (кроме Transform) со всех дочерних объектов рекурсивно.

Внимание: Этот метод использует GameObject.DestroyImmediate, что рекомендуется использовать только в режиме редактора. Использование в рантайме может привести к проблемам и утечкам памяти.

Аргументы
objКорневой Transform, начиная с которого происходит удаление.
excludeTypesМассив типов компонентов, которые не следует удалять.
