Свойства

bool WillUnsubscribeAfterFiring [get, set]
 By default, WaitForSpineEvent will unsubscribe from the event immediately after it fires a correct matching event. If you want to reuse this WaitForSpineEvent instance on the same event, you can set this to false.

Подробное описание

Use this as a condition-blocking yield instruction for Unity Coroutines. The routine will pause until the AnimationState fires an event matching the given event name or EventData reference.
